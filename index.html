OpenSage: Self-programming Agent Generation Engine

An AI-centric framework that enables LLMs to self-create agent topology, synthesize toolsets, and manage hierarchical memory for complex real-world tasks.

Paper GitHub Docs Discord
Core Capabilities
Self-Creating Topology
AI autonomously designs and restructures multi-agent topologies at runtime based on task requirements.
Tool Synthesis
Dynamically generates and composes toolsets on-the-fly, enabling agents to extend their own capabilities.
Hierarchical Memory
Structured memory management across abstraction levels for sustained context over long-horizon tasks.
